# Django Forum
Форум, написанный на Django 4.1.1.

## Описание проекта
Проект написан в образовательных целях. Используемые технологии:
- django 4.1.1
- sqlite [Возможно изменится]
- Bootstrap [Frontend]

## Фишки проекта
- [ ] Регистрация пользователей
- [ ] Роли пользователей
- [ ] Защита от ботов
- [ ] Форумы по категориям

Все задачи и их прогресс будут заведены в [бэклог проекта](https://trello.com/b/l74Uicj1/forum-%D0%B1%D1%8D%D0%BA%D0%BB%D0%BE%D0%B3)
